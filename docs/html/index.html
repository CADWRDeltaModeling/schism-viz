This repo has a number of example notebooks which give simple examples. As those functionality evolves, the scripts in schism_viz house common functions

The notebooks found in the github repository contain the following exercises:

conda env create --name schism_viz -f environment.yml

This creates a new environment called “schism_viz” which contains the necessary packages to run through these modules.
